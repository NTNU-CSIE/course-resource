## 📦 建議格式

為了因應各種形式的資源，所有格式都是可接受的。

但是，基於方便維護、減少大小的考量，建議原始文件資料使用 Markdown 格式。

> 對於數學相關的資料，可以使用 `$O(n \times m)$` 來表現出 $O(n × m)$。
